What stays true

Three principles, no exceptions.

01 — Used first

Built to be used.

Each tool earns its place by solving a real task before the source is opened. Nothing ships as a demo.

02 — Shows its work

Answers carry receipts.

If a tool tells you something, it tells you where that came from — the article, the file, the run.

03 — Open by default

Source is the spec.

Every tool is public on GitHub. Read it, run it, fork it. The code is the documentation of record.
