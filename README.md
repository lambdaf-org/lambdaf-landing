# lambdaforge-landing

The landing page for **Lambdaforge** — [lambdaf.org](https://lambdaf.org).

A single static page, no build step: `index.html` plus the brand mark, favicon, and contributor
avatars under `people/`. Fonts load from Google Fonts (IBM Plex Mono, Spectral, Zilla Slab); everything
else is local. To preview, open `index.html` in a browser.
